# Website Profil Personal

Website profil responsif yang dibuat dengan HTML, CSS, dan JavaScript murni.

## 🎨 Fitur

- **Responsif & Modern**: Desain yang indah dan berfungsi di semua perangkat
- **Smooth Scrolling**: Navigasi yang mulus antar bagian
- **Dark Theme Ready**: Mudah dikustomisasi
- **Animasi Menarik**: Efek hover dan scroll animations
- **Mobile Menu**: Hamburger menu untuk mobile devices
- **SEO Friendly**: HTML semantik yang baik

## 📁 Struktur File

```
website-profil/
├── index.html      # File HTML utama
├── styles.css      # Stylesheet
├── script.js       # JavaScript interaktif
├── cv.pdf          # File CV (tempat Anda upload CV)
└── README.md       # File dokumentasi ini
```

## 🚀 Cara Menggunakan

1. **Ganti informasi pribadi**:
   - Buka `index.html`
   - Ganti "Nama Anda" dengan nama Anda
   - Ganti deskripsi profil, pengalaman, keahlian, dll.

2. **Ganti foto profil**:
   - Ganti link foto di baris: `<img src="https://via.placeholder.com/200" alt="Foto Profil">`
   - Gunakan foto lokal atau URL gambar Anda

3. **Tambah portfolio Anda**:
   - Ganti judul, deskripsi, dan teknologi di bagian Portfolio
   - Ubah link gambar portfolio

4. **Informasi kontak**:
   - Ganti email, telepon, dan lokasi
   - Update link social media

5. **CV/Resume**:
   - Buat file `cv.pdf` dan letakkan di folder yang sama
   - Link download akan otomatis bekerja

## 🎯 Kustomisasi Warna

Edit variabel di bagian `:root` di file `styles.css`:

```css
:root {
    --primary-color: #007bff;        /* Biru */
    --secondary-color: #6c757d;      /* Abu-abu */
    --dark-color: #343a40;           /* Gelap */
    --light-color: #f8f9fa;          /* Terang */
    /* dst... */
}
```

## 📱 Fitur Responsif

- ✅ Desktop (1200px+)
- ✅ Tablet (768px - 1199px)
- ✅ Mobile (480px - 767px)
- ✅ Xtra Small (< 480px)

## 🔗 Bagian-bagian Website

1. **Navbar** - Menu navigasi sticky di atas
2. **Hero/Beranda** - Selamat datang dengan CTA buttons
3. **Tentang** - Deskripsi tentang Anda dan statistik
4. **Keahlian** - Grid kartu untuk menampilkan skill
5. **Portfolio** - Showcase proyek-proyek Anda
6. **Pengalaman & Pendidikan** - Timeline CV
7. **Kontak** - Form kontak dan info kontak
8. **Social Links** - Link ke social media
9. **Footer** - Copyright

## 💡 Tips

- Gunakan font yang bagus dengan Google Fonts jika ingin menambah
- Pastikan gambar di-optimize untuk web (tidak terlalu besar)
- Ganti icon Font Awesome sesuai kebutuhan
- Test di mobile menggunakan browser DevTools
